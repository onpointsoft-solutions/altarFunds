Sanctum Church Management System - Linux Version

Installation Instructions:
------------------------

Option 1: Automatic Installation (Recommended)
-----------------------------------------------
1. Run: chmod +x install.sh && ./install.sh
2. Follow the on-screen instructions
3. Desktop shortcut will be created automatically

Option 2: Portable Mode
-----------------------
1. Run: chmod +x sanctum.sh && ./sanctum.sh
2. No installation required
3. Can be run from any location

System Requirements:
-------------------
- Linux (Ubuntu, Fedora, Arch, etc.)
- Java Runtime Environment (JRE) 11 or higher
- 100MB free disk space
- 2GB RAM minimum

Features:
--------
- User authentication with role-based access
- Church member management
- Financial tracking and reporting
- Event scheduling and management
- Communication tools
- Full-screen immersive interface
- Built-in SQLite database (no external database required)

Uninstallation:
---------------
Run: chmod +x uninstall.sh && ./uninstall.sh

Troubleshooting:
---------------
- If the application doesn't start, ensure Java 11+ is installed
- For portable mode, ensure you have execute permissions
- For installed mode, ensure proper file permissions
